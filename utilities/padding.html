      Responsive prefixes

      Add a responsive breakpoint prefix separated with a : to target a utility at a responsive breakpoint and higher, following a mobile-first methodology.

      Example

      ---
layout: iframe
permalink: utilities/html/padding-responsive
---
<div class="vads-u-padding--2 medium-screen:vads-u-padding--5 large-screen:vads-u-padding--9" style="background-color:#112e51; color: white">
  <h1 class="vads-u-margin--0">This is a demo</h1>
  <p class="vads-u-margin--0">Use the tabs to see how the responsive padding utility works</p>
</div>
      breakpoint-name width
      xsmall-screen 320px
      small-screen 481px
      medium-screen 768px
      small-desktop-screen 1008px
      large-screen 1201px
